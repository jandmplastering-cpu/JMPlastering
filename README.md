## Publish on GitHub Pages
Upload all files to a GitHub repository, then enable GitHub Pages from the repository settings.


## Reviews setup

The website now has a dedicated `reviews.html` page and buttons linking customers to Google Reviews.

Because Google does not provide a simple free automatic website embed without a Business Profile/widget setup, the reviews page includes a ready-made widget area. To make new reviews appear automatically on the website:

1. Set up or claim the J&M Plasters Google Business Profile.
2. Copy the direct “Ask for reviews” link from Google Business Profile and replace the Google search link in `reviews.html` and `index.html`.
3. Create a Google Reviews widget using Elfsight, Trustindex, SociableKIT, or another widget provider.
4. Paste the widget code into the marked widget area in `reviews.html`.

Until the direct review link is added, the review buttons open a Google search for “J&M Plasters Manchester reviews”.
